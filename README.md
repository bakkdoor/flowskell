Flowskell - Fluxus clone in Haskell
===================================

Flowskell allows you to write visual animations using the Scheme language. It is driven by OpenGL and the husk-scheme interpreter.

Run Flowskell
-------------

    cabal configure
    cabal build
    cabal install
    flowskell examples/recursion.scm
